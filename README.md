# 360
A multi-tool browser.
Our Reddit: https://www.reddit.com/r/360Search/

360 is a multi-tool browser (search engine) that has more tools to out perform any browser through convienently organized UI's all ready so your work doesn't need to be done on like 50+ tabs...
With many search engines and browsers losing trust from core users, this is a great opportunity for not only you to switch to a fully private, fully open source search engine, but also for US! Using 360 supports 360Digital, Co. and every contribution counts! What other browser helps people for helping them? Switch to 360 now! (if you want. we're humble and will not be pushy)
